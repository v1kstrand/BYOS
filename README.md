# BYOS (Bring Your Own State)

Standalone research prototype for long-context language modeling with an explicit persistent state memory `S`.

## Layout

- Model + predictor: `BYOS/byos_prototype.py`
- Training loop: `BYOS/byos_train.py`
- Data: `BYOS/byos_data.py`
- Eval/generation: `BYOS/byos_eval.py`
- Monitoring: `BYOS/byos_monitor.py`
- Notebook: `BYOS/byos_v0_5.ipynb`
- Configs: `BYOS/cfg_byos*.yaml`
